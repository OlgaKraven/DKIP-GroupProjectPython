# Инвестиционный Аналитик ИС (InvestCalc)  
## Групповой проект

Учебный проект по оценке экономической эффективности информационных систем.

## Навигация

- [Guide — руководство по проекту](guide.md)
- [Правила оформления документации](doc-guidelines.md)

## Компоненты проекта

- REST API (`/calc/local`, `/calc/cloud`, `/calc/sensitivity`, `/compare`)
- Расчётный модуль (TCO, ROI, Payback)
- Анализ чувствительности (±20%)
- Отчёт (HTML/PDF)
- Docker-развёртывание
