# Use Cases — InvestCalc

## UC-01 — Локальный расчёт
1. Пользователь отправляет POST `/calc/local` с данными проекта.
2. Система валидирует данные.
3. Система рассчитывает показатели.
4. Система возвращает JSON с результатами.

## UC-02 — Облачный расчёт
Аналогично UC-01, но с параметрами облачной модели.

## UC-03 — Анализ чувствительности
1. Пользователь вызывает `/calc/sensitivity`.
2. Система формирует три набора расчётов: -20%, базовый, +20%.
3. Система возвращает результаты.

## UC-04 — Сравнение сценариев
1. Пользователь вызывает `/compare`.
2. Система сравнивает локальный и облачный сценарии.
3. Система возвращает сравнительную таблицу.
