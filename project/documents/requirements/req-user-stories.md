# User Stories — InvestCalc

## US-01
Как аналитик, я хочу ввести данные по ИС, чтобы получить расчёт TCO, ROI и Payback.

## US-02
Как аналитик, я хочу сравнить локальный и облачный сценарии, чтобы выбрать оптимальный вариант.

## US-03
Как аналитик, я хочу увидеть анализ чувствительности (±20%), чтобы оценить риски проекта.

## US-04
Как аналитик, я хочу получить HTML-отчёт, чтобы приложить его к аналитическим материалам для заказчика.

## US-05
Как DevOps-специалист, я хочу запускать сервис через Docker, чтобы упростить развёртывание.
