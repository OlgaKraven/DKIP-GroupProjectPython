# Функциональные требования — InvestCalc

## 1. Расчётные функции
- Расчёт TCO.
- Расчёт ROI.
- Расчёт срока окупаемости (Payback Period).
- Анализ чувствительности (±20% ключевых параметров).
- Сравнение локального и облачного сценариев.

## 2. API
Реализовать эндпоинты:
- `POST /calc/local`
- `POST /calc/cloud`
- `POST /calc/sensitivity`
- `POST /compare`

Все запросы и ответы — JSON.

## 3. Отчёт
Система должна формировать HTML-отчёт с:
- исходными данными;
- результатами расчётов;
- анализом чувствительности;
- выводами.
