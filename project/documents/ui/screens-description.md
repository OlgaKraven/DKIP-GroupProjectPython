# Описание экранов — InvestCalc

## Экран 1 — Ввод данных
Поля ввода CAPEX, OPEX, обучение, экономия, доходность, период.  
Кнопка «Рассчитать локальный/облачный сценарий».

## Экран 2 — Результаты
Отображение TCO, ROI, Payback.  
Кнопка «Показать отчёт».

## Экран 3 — Чувствительность
Таблица -20% / базовый / +20%.  
Визуальное сравнение.

## Экран 4 — Сравнение сценариев
Сравнительная таблица локального и облачного вариантов.
