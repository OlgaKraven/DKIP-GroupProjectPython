# Test Cases — InvestCalc

## TC-01 — Локальный сценарий
Шаги: POST `/calc/local` с корректными данными.  
Ожидание: статус 200, корректные значения TCO, ROI, Payback.

## TC-02 — Облачный сценарий
POST `/calc/cloud`.  
Ожидание: корректные результаты.

## TC-03 — Чувствительность
POST `/calc/sensitivity`.  
Ожидание: три набора значений (minus20/base/plus20).

## TC-04 — Сравнение сценариев
POST `/compare`.  
Ожидание: JSON с двумя сценариями и сравнительными полями.

## TC-05 — Неверные данные
Передать строку вместо числа.  
Ожидание: код ошибки и сообщение о некорректном типе.
