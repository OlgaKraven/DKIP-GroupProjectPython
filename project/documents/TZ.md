# Техническое задание (ТЗ)  
## Мини-информационная система «Инвестиционный Аналитик ИС (InvestCalc)»

## 1. Общие сведения
- Название системы: InvestCalc
- Назначение: оценка экономической эффективности ИС
- Заказчик (легенда): агентство цифрового консалтинга DataForge

## 2. Назначение и цели
Система рассчитывает TCO, ROI, Payback Period, выполняет анализ чувствительности ±20% и сравнение локального и облачного сценариев для выбранной ИС.

## 3. Функциональные требования (кратко)
- Принимать входные данные (CAPEX, OPEX, обучение, экономия, доходность, период).
- Реализовать эндпоинты:
  - `POST /calc/local`
  - `POST /calc/cloud`
  - `POST /calc/sensitivity`
  - `POST /compare`
- Формировать HTML-отчёт.

## 4. Нефункциональные требования (кратко)
- Запуск через Docker.
- Документация в Markdown.
- Простая модульная архитектура.

## 5. Входные/выходные данные
Вход: JSON с параметрами проекта.  
Выход: JSON с показателями TCO, ROI, Payback, сценариями чувствительности.

## 6. Этапы выполнения
1. Анализ предметной области и модель.
2. Архитектура и первый эндпоинт.
3. Чувствительность, отчёт, Docker.
4. Финальная сборка и презентация.

## 7. Порядок приёмки
Проект считается принятым при:
- работоспособности всех эндпоинтов;
- корректных расчётах;
- наличии Docker-развёртывания;
- наличии полного комплекта документации;
- успешной защите.
