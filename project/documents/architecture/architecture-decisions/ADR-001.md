# ADR-001 — Выбор архитектурного стиля

## Контекст
Нужна простая архитектура для учебного сервиса InvestCalc.

## Решение
Использовать слойную архитектуру:
- API слой;
- сервисный слой (расчёты);
- слой моделей;
- слой отчётности.

## Последствия
+ Простота понимания студентами  
+ Лёгкость тестирования  
− Для сложных систем может потребоваться дополнительная декомпозиция
